How It Works

1

Download and unzip

Click the download button above. Unzip the file anywhere on your computer.

2

Load the browser extension

Firefox: Go to about:debugging → Load Temporary Add-on → select manifest.json from the extension folder.
Chrome: Go to chrome://extensions → Developer mode ON → Load unpacked → select the extension folder.

3

Click the Handoff icon on any AI chat

Works on ChatGPT, Claude, Gemini, Grok, DeepSeek, Perplexity, Mistral, Kiro, Copilot, and 30+ more. Also works on creative AI tools like Runway, Midjourney, Suno.

4

Click Extract → Copy → Paste into another agent

Choose "Summary" to save tokens or "Full" for complete context. Or download as a file to attach. The receiving agent picks up exactly where you left off.
